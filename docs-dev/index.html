The Network Monitor is a dynamic platform plugin for Homebridge. This plugin will report the performance of the network by publishing the results of ping operations as Homekit Carbon Dioxide Sensor services.

There are several configutation settings for the plugin. Refer to the README.md that is included with the plugin.
